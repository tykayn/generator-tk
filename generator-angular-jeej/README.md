## Getting Started
you need yeoman from NPM. To install it in command line, run this:
```sh
$ npm i -g yo
```
to run the generator, use yo.
```sh
$ yo angular-jeej
```
You will be asked a name for your webapp.
After all is set up, enjoy your new app.
